# Springboot-Webservice


## 개발 인원
- **개인 프로젝트**


## 개발 기간
- **2024.1.15 ~**


## 주요 기능
- OAuth2.0 소셜 로그인(구글, 네이버) 기능
- 일반 회원가입, 로그인, 회원정보 수정 기능
- 게시글, 댓글 CRUD 기능
    - 게시글 사진 업로드 기능
- 게시글, 댓글 추천, 비추천 기능
- 검색 기능(제목, 내용, 작성자, 제목+내용)
- 페이징 처리

    
## 기술 스택
- **Java**
- **Thymeleaf**
- **Spring Boot**
- **JPA**
- **Spring Security**
- **MySQL**
- **IntelliJ**
